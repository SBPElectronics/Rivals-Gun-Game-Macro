# Rivals-Gun-Game-Macro

A Python-based AFK macro tool for Roblox with built-in gamepass verification. Users must own the required gamepass before the bot unlocks, acting as a lightweight license system. Once verified, the bot automatically moves, shoots, jumps, slides, and melees using simulated keypresses, keeping your character active while AFK. All hotkeys are fully configurable through a clean GUI.
